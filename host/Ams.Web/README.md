## Welcome!

Thank you for creating a new project with bit platform! We appreciate your trust in our platform and are excited to see what you'll build.

This project gets generated by bit-bp template v-10.1.0 using the following command
```bash
    dotnet new bit-bp
            --name Ams.Web
            --database SqlServer
            --filesStorage AzureBlobStorage
            --notification false
            --appInsights
            --cloudflare false
```
