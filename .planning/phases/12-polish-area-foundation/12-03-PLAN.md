  <action>
Create `PickupMatchingService` in `Ams.Workstation.Server.Services`.

**IMPORTANT:** Use `AsrProcessor.TranscribeBufferAsync` exclusively (Whisper.NET in-proc). Nemo ASR is obsolete. The service does NOT need IAsrService -- call the static AsrProcessor directly.

Public methods:

`MatchPickupAsync(string pickupFilePath, IReadOnlyList<HydratedSentence> flaggedSentences, CancellationToken ct)`:
1. Decode pickup file: `AudioProcessor.Decode(pickupFilePath)`
2. Detect silence for session file segmentation: `AudioProcessor.DetectSilence(pickupBuffer, new SilenceDetectOptions { NoiseDb = -45.0, MinimumDuration = TimeSpan.FromSeconds(1.5) })`
3. Derive segments from silence intervals (gaps between silences are speech segments). Private helper `DeriveSegments(AudioBuffer buffer, IReadOnlyList<SilenceInterval> silences)` -- compute start/end from silence boundaries, minimum segment duration 0.3s to filter noise.
4. For each segment:
   a. Trim segment: `AudioProcessor.Trim(pickupBuffer, start, end)`
   b. Prepare for ASR: `AsrAudioPreparer.PrepareForAsr(segBuffer)`
   c. Run ASR: `await AsrProcessor.TranscribeBufferAsync(asrReady, null, ct)`
   d. Extract full text from ASR response tokens
   e. Fuzzy match against `flaggedSentences` using `LevenshteinMetrics.Similarity` on normalized text (lowercase, trim whitespace)
   f. Accept match if confidence > 0.5 (50% similarity threshold)
   g. Create `PickupMatch` record
5. Return `List<PickupMatch>` sorted by PickupStartSec

`MatchSinglePickupAsync(string pickupFilePath, HydratedSentence targetSentence, CancellationToken ct)`:
- Simplified version for individual pickup files (one per sentence)
- Run ASR on entire file, create match with the target sentence directly
- Still compute confidence via Levenshtein for the user's reference

Private `string NormalizeForMatch(string text)` -- lowercase, collapse whitespace, remove punctuation. Use existing `TextNormalizer` patterns from Ams.Core.Common if appropriate, otherwise inline simple normalization.

Constructor: no DI dependencies needed (all methods use static processors).

Create `PolishService` in `Ams.Workstation.Server.Services`. This is the main orchestrator that ties pickup matching, staging, splicing, and undo together.

Constructor DI: `BlazorWorkspace`, `StagingQueueService`, `UndoService`, `PickupMatchingService`.

Public methods:

`ImportPickupAsync(string pickupFilePath, IReadOnlyList<HydratedSentence> flaggedSentences, CancellationToken ct)`:
- Delegates to `PickupMatchingService.MatchPickupAsync`
- Returns `List<PickupMatch>` for UI to display

`StageReplacement(string chapterStem, PickupMatch match, string pickupFilePath, double crossfadeSec = 0.030, string curve = "tri")`:
- Creates a `StagedReplacement` record from the match (generate GUID for Id, set status Staged, timestamp)
- Calls `StagingQueueService.Stage(replacement)`
- Returns the created StagedReplacement

`ApplyReplacementAsync(string replacementId, CancellationToken ct)`:
1. Get staged item from `StagingQueueService`
2. Load chapter audio from workspace: `_workspace.CurrentChapterHandle.Chapter.Audio.Current.Buffer`
3. Save original segment via `UndoService.SaveOriginalSegment(...)`
4. Decode pickup audio: `AudioProcessor.Decode(item.PickupSourcePath)`
5. Trim pickup to boundaries: `AudioProcessor.Trim(pickupBuffer, start, end)`
6. Call `AudioSpliceService.ReplaceSegment(chapterBuffer, item.OriginalStartSec, item.OriginalEndSec, pickupTrimmed, item.CrossfadeDurationSec, item.CrossfadeCurve)`
7. Calculate timing delta: `(pickupDuration - originalDuration)` for downstream timing shift
8. Update staging status to Applied: `StagingQueueService.UpdateStatus(replacementId, ReplacementStatus.Applied)`
9. Return the new AudioBuffer and timing delta

`RevertReplacementAsync(string replacementId, CancellationToken ct)`:
1. Get undo record from `UndoService`
2. Load original segment: `UndoService.LoadOriginalSegment(replacementId)`
3. Re-splice: use `AudioSpliceService.ReplaceSegment` to put the original back
4. Update status to Reverted: `StagingQueueService.UpdateStatus(replacementId, ReplacementStatus.Reverted)`
5. Return restored buffer and negative timing delta

`GetStagedReplacements(string chapterStem)`:
- Returns `StagingQueueService.GetQueue(chapterStem)`
