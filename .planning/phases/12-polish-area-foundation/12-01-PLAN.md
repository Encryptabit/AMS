  <files>host/Ams.Core/Audio/AudioSpliceService.cs</files>
  <action>
Create `AudioSpliceService` as a static class in `Ams.Core.Audio` namespace (following AudioProcessor pattern -- static, stateless).

Implement `ReplaceSegment(AudioBuffer original, double startSec, double endSec, AudioBuffer replacement, double crossfadeSec = 0.030, string curve = "tri")`:
1. If `replacement.SampleRate != original.SampleRate`, resample replacement via `AudioProcessor.Resample(replacement, (ulong)original.SampleRate)`
2. Trim `before` segment: `AudioProcessor.Trim(original, TimeSpan.Zero, TimeSpan.FromSeconds(startSec))`
3. Trim `after` segment: `AudioProcessor.Trim(original, TimeSpan.FromSeconds(endSec))`
4. Clamp crossfade duration: `min(crossfadeSec, before.Duration * 0.3, replacement.Duration * 0.3)` -- per Pitfall 2 in research
5. Crossfade `before` + `replacement` using private `Crossfade` method
6. Clamp crossfade again for second join: `min(crossfadeSec, joinedResult.Duration * 0.3, after.Duration * 0.3)`
7. Crossfade result + `after`
8. Return final buffer

Private `Crossfade(AudioBuffer a, AudioBuffer b, double durationSec, string curve)`:
- If durationSec <= 0.001 or either buffer is empty, fall back to `AudioBuffer.Concat(a, b)`
- Build filter spec using `FfFilterGraphRunner.Apply` with two labeled inputs ("a", "b"):
  `[a]aformat=sample_fmts=flt[af];[b]aformat=sample_fmts=flt[bf];[af][bf]acrossfade=d={durationSec:F6}:c1={curve}:c2={curve}[out]`
- Use `FormattableString.Invariant()` for locale-safe decimal formatting

Note: `FfFilterGraphRunner.GraphInput` is `internal` -- the service lives in the same assembly (Ams.Core) so it has access. Use `new FfFilterGraphRunner.GraphInput("a", a)` directly.

Add XML doc comments on the public method. Add a helper `DurationSeconds(AudioBuffer b)` to compute `(double)b.Length / b.SampleRate`.
