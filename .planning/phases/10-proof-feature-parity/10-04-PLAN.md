**Add state for view and report:**
```csharp
private string _currentView = "playback"; // Start with existing playback view
private ChapterReport? _report;

private void LoadReport(string chapterName)
{
    // Direct service call — NO HttpClient
    var hydrate = Workspace.CurrentChapterHandle?.Chapter.Documents.HydratedTranscript;
    if (hydrate != null)
    {
        _report = ProofReportService.BuildReport(chapterName, hydrate);
    }
}

private void SwitchView(string view)
{
    _currentView = view;
    StateHasChanged();
}
```

**Load report in OnParametersSetAsync** (after SelectChapter and LoadSentencesAsync):
```csharp
// After existing LoadSentencesAsync call:
LoadReport(decodedName);
```

**Conditionally render views:**
```razor
@if (_currentView == "playback")
{
    @* Existing playback view with WaveformPlayer and SentenceList (current BitGrid) *@
}
else
{
    @* Errors view *@
    <ErrorsView Report="_report" OnPlaySentence="HandlePlayFromErrors" />
}
```

**Add handler for errors view play action (bridges SentenceReport -> seek + switch view):**
```csharp
private async Task HandlePlayFromErrors(SentenceReport sentence)
{
    _currentView = "playback";
    StateHasChanged();
    if (_waveformPlayer != null)
    {
        await _waveformPlayer.SeekTo(sentence.StartTime);
    }
}
```

Note: This handler takes `SentenceReport` (from ErrorsView) not `SentenceViewModel` (from SentenceList).

        ?? Enumerable.Empty<SentenceReport>();

    private double ParseWer(string wer)
    {
        if (double.TryParse(wer.TrimEnd('%'), out var v)) return v;
        return 0;
    }

    }

    }
}
