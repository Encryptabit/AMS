### Cache Implementation

**Dual-layer caching strategy verified:**

1. **ASR Cache** (PickupMatchingService.cs lines 266-309):
   - Key: SHA256 of `fullpath|size|lastwrite`
   - Location: `{workDir}/.polish/pickups/{hash}.asr.json`
   - Stores raw ASR response for instant re-import

2. **MFA Cache** (PickupMfaRefinementService.cs lines 338-397):
   - Key: SHA256 of `fullpath|size|lastwrite|sentenceIds|normalizedBookText`
   - Location: `{workDir}/.polish/pickups/{hash}.mfa.json`
   - Stores refined PickupMatch records with MFA timings
   - **Critical design:** Cache key includes sentence context (IDs + BookText), preventing stale timing reuse when same audio is matched to different sentences

### Needleman-Wunsch Alignment Strategy

**Synthetic bookIdx approach verified (lines 229-304):**

1. Build flat word list across all matched sentences
2. Assign synthetic "bookIdx" values: sentence 0 words → indices 0..N-1, sentence 1 words → N..M-1, etc.
3. Track `sentenceRanges: (sentenceId, startIdx, endIdx)` for each sentence
4. Create `SentenceTarget[]` with callbacks that capture `(start, end, duration)` for each sentence
5. Call `MfaTimingMerger.MergeAndApply()` with:
   - `textGridWords` (filtered: no silence/sp/sil tokens)
   - `GetBookToken(idx)` function from flat word list
   - `wordTargets` (empty — only sentence timings needed)
   - `sentenceTargets` with callbacks
6. Extract MFA timings from `sentenceTimings` dictionary populated by callbacks
7. Build refined PickupMatch records with MFA boundaries, falling back to ASR for unmatched sentences

**Result:** Robust handling of word count mismatches from contractions, hyphen splits, OOV tokens — identical strategy to MfaTimingMerger used in chapter alignment.

### Graceful Degradation

**Fallback points verified:**

1. **Empty matches:** Line 51-52, early return if no matches to refine
2. **Empty lab content:** Lines 89-93, return ASR matches if BookText normalization produces no usable words
3. **Align failure:** Lines 157-161, return ASR matches if MFA align exits non-zero
4. **Missing TextGrid:** Lines 165-169, return ASR matches if TextGrid file not found after alignment
5. **Global exception:** Lines 191-195, catch any unexpected error and return ASR matches (preserves OperationCanceledException)

**Design principle:** MFA refinement is an enhancement, never a blocker. Pickup import workflow succeeds with ASR-only timings if MFA is unavailable or fails.
