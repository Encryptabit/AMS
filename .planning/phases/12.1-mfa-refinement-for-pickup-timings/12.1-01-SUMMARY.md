# Phase 12.1 Plan 01: MFA Refinement for Pickup Timings Summary

**MFA forced alignment post-processing for pickup matches via PickupMfaRefinementService, producing phoneme-accurate sentence boundaries from TextGrid intervals mapped through Needleman-Wunsch alignment**

## Accomplishments
- Created PickupMfaRefinementService (445 lines) that runs MFA forced alignment on pickup audio to replace rough ASR token boundaries with phoneme-accurate sentence timings
- Integrated MFA refinement into PickupMatchingService.MatchPickupAsync so all returned PickupMatch records have MFA-refined boundaries
- Added MFA result caching keyed on audio identity + sentence context for instant re-import
- Established graceful fallback: any MFA failure returns original ASR timings without blocking the import workflow

## Files Created/Modified
- `host/Ams.Workstation.Server/Services/PickupMfaRefinementService.cs` - MFA-based timing refinement service: builds temp corpus, runs MFA validate/G2P/align, parses TextGrid, maps intervals to sentences via MfaTimingMerger, caches results
- `host/Ams.Workstation.Server/Services/PickupMatchingService.cs` - Added PickupMfaRefinementService dependency and RefineWithMfaAsync call after ASR matching
- `host/Ams.Workstation.Server/Program.cs` - DI registration for PickupMfaRefinementService and MFA background warmup at startup
- `host/Ams.Core/AssemblyInfo.cs` - Added InternalsVisibleTo("Ams.Workstation.Server") for MfaService/MfaProcessSupervisor access

## Decisions Made
- **InternalsVisibleTo over public API wrapper:** Added assembly attribute rather than creating thin public wrappers on MfaService/MfaProcessSupervisor. Simpler, and the workstation is a first-party host that legitimately needs MFA internals.
- **MFA background warmup at startup:** Added MfaProcessSupervisor.TriggerBackgroundWarmup() in Program.cs so conda environment is pre-warmed before first pickup import.
- **Cache key includes sentence context:** SHA256 of path|size|lastwrite|sentenceIds|normalizedBookText prevents stale MFA timings when the same audio is re-matched with different sentences.
